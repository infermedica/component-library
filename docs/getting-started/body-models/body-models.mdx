# Body Models

Infermedica's SVG Body Models are now available. The SVG Body Models are a crucial component of our application, providing clear and detailed visual representations of selected symptoms in the human body. Highlighting symptoms associated with specific body parts is possible thanks to the symptom-to-body-part mapping.
This mapping can be found below in the section **Body Parts Mapping**.

## Usage

Choose the age group of interest from the side panel. Use the Docs tab to view all SVG models dedicated to that group.
<img src={SCR1} alt="" />

Familiarize yourself with the CSS styling information, as you will be copying the SVG without styles.
<img src={SCR2} alt="" />

Use the "Show code" option to display the SVG model code for the selected frame.
<img src={SCR3} alt="" />

Use the "Copy" option to download the SVG body model code.
<img src={SCR4} alt="" />

## Body parts mapping

Each SVG path in this project has an assigned body part, which can be accessed via the data-bodypart attribute. This attribute allows easy identification and manipulation of individual body parts within the SVG code.
With the data-bodypart attribute, it is possible to easily and intuitively operate on different body parts represented in SVG, facilitating the creation of interactive graphics.

Below is a JSON structure that enables the mapping of symptoms to body parts, allowing for the highlighting of body parts affected by the given symptoms. Using this JSON structure, you can easily manage and visualize the symptoms on the SVG representation of the body, enhancing interactivity and user experience.
