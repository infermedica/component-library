# 0.6.0 (February 22, 2023)

* use CSS logical properties instead of directional ones
* remove UiMegaMenu components. Use UiHorizontalPaging instead

## 🚀 Features
* add export types in index
* helper for css logical properties
* editable global tokens on subpags of the design system
* add update CSS Variables script
* add CSS variables to SVG illustrations

## 🧹 Refactor
* withDefaults and type props declaration in templates
* withDefaults and type props declaration in organisms
* withDefaults and type props declaration in molecules
* withDefaults and type props declaration in atoms

## 🐛 Fixes
* multiple answer item has button inside label
* elements with negative tabindex will be recognized as focusable

## 🧹 Chores
* run only test on pre-push hook
* remove unused *.js files from scripts

## 📄 Docs
* add custom table component and toolbar addon for design system
* move inline styles from stories to scss files
* auto generate release notes
