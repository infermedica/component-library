# Migration Guide

v0.5.1 use SASS @use instead @import

<PageOutline/>

## SASS @use
From v0.5.1 we use SASS `@use` instead `@import` because `@import` will be dropped by October 2022.

**Before**

```scss
@import "../../../styles/mixins/mixins";
@import "../../../styles/functions/functions";

.ui-component {
  $element: component;
  @include font($element, button-1);
  padding: css-var($element, padding, var(--space-12) var(--space-32));
}
```

**After**
```scss
@use "../../../styles/functions";
@use "../../../styles/mixins";

.ui-component {
  $element: component;
  @include mixins.font($element, button-1);
  padding: functions.var($element, padding, var(--space-12) var(--space-32));
}
```
