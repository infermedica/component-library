# v0.3.1 Migration Guide

## UiTextarea
From v0.3.1, textarea HTML element is wrapped in a div with class `ui-textarea`. i.e. to focus textarea you must select `textarea`

<Code code={`
<template>
 <UiTextarea
  ref="textarea"
 />
</template>

<script setup>
 import { ref, nextTick, onMounted } from 'vue';

 const textarea = ref(null)
 onMounted(async()=> {
  await nextTick();
  textarea.value?.$el
   .querySelector('textarea') // select textarea
   .focus();
 })
</script>
`}/>
