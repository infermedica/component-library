# Fade transition
This visual effect that smoothly changes the opacity of an element, creating a gradual appearance or disappearance.<br/>
This transition effect can be applied to various elements, such as text, images, or entire sections of a web page, adding a subtle and visually appealing animation to the user interface.

## How to use
To use ```fade``` transition, just surround your component with ```Transition``` <a href="https://vuejs.org/guide/built-ins/transition.html" target="_blank">Vue build-in component</a> with attribute ```name="fade"```
