# Keyboard focus
Focus state in Infermedica Component Library is hidden for mouse users. To hide focus we ass to `body` element class `focus-hidden`.
After adding this directive to the component you can handle `focus-hidden` on the `body` and show hide focus for mouse navigation.

## How to use
* `v-keyboard-focus` - detect mouse / keyboard navigation
```html
<button v-keyboard-focus>...</button>
```

### focusElement helper
In some cases you need to set focus to element. This focus should be always visible. To do this you can use `focusElement` helper.

```js
import { focusElement } from '@infermedica/component-library/helpers';

const closeButton = ref(null);
const handleModalOpen = () => {
  focusElement(closeButton.value?.$el);
};
```
