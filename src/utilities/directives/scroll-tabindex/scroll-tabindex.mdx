# Scroll tabindex
This directive add tabindex attribute to element only when content can be scrolled. Now element is focusable and you can control scrolling with keyboard

## How to use
* `v-scroll-tabindex` - add tabindex attribute

<Code code={`
<div
  class="wrapper"
  v-scroll-tabindex
>
  <div class="wrapper__scrollable-content">
    ...
  </div>
</div>
`}/>
