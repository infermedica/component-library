## How to use
UiLoader has 3 transition types: `if`, `show` and `opacity`.
Each of these transition types will find its use in different situation.

### If Transition Type
`if` transition type use `<Transition/>` and `v-if` directive.
Use this transition type when you want to hide sensitive content while is loading.

<Alert>
  <p>You can put to <code>UiLoader</code> multi elements because we wrap it in to <code>div</code> to allow use <code>Transition</code></p>
</Alert>

<Canvas>
  <Story id="molecules-loader--if-transition-type"/>
</Canvas>

### Show Transition Type
`show` transition type use our custom `<UiLoaderTransition>`.
Use this transition type allow to fetch data onMounted event while is loading.

<Canvas>
  <Story id="molecules-loader--show-transition-type"/>
</Canvas>

### Opacity Transition Type
`opacity` transition type use our custom `<UiLoaderTransition>`.
Use this transition type allow to keep size of loading content while is loading.
It's useful for buttons with loading state.
